## 🌌 What Is ExoPlanet Companion?

ExoPlanet Companion is an interactive space exploration web app that lets users browse and discover over **6,000 real confirmed exoplanets** from NASA's Exoplanet Archive.

For each planet, an AI model generates a vivid simulation of what that world might be like — including its appearance, sky, atmosphere, potential for life, and unique scientific insights.

Every planet a user explores is saved to their personal **Explorer Journal**, powered by Backboard’s persistent memory system. This creates a living record of their journey across the cosmos.

---

## ✨ Features

- **🔭 Real NASA Data** — Fetches live exoplanet data (radius, mass, orbital period, temperature, and distance)
- **🤖 AI-Generated Planet Simulations** — Open-source language models generate scientifically grounded yet imaginative world descriptions
- **📓 Persistent Explorer Journal** — Backboard memory system stores every explored planet across sessions
- **🔍 Search & Discover** — Search planets by name or explore random selections from thousands of real worlds
- **✓ Visited Tracking** — Visited planets are marked for easy exploration history
- **🌠 Immersive Space UI** — Animated starfield background and cinematic dark-space design
